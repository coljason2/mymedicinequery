# 查藥爬蟲系統
## 環境
* Spring Boot 
* Heroku

## 專案說明
* 以Spring Boot開發串接Heroku，主要從嘉鏵藥品網站將藥品相關資訊爬下來。
* 功能介紹：
    * 查詢藥品資訊
    * 7-11上傳列印串接
    * PDF/CSV匯出
    * 健保網頁查詢
    * Swagger API 支援
    * 支援多藥品公司查詢
