# 查藥爬蟲系統
## 環境
* Spring Boot 
* GCP
* Cloudflare

## 專案說明
* 以Spring Boot開發部屬GCP註冊免費Domain，主要從大盤商藥品網站將藥品相關資訊爬下來。
* 功能介紹：
    * 查詢藥品資訊
    * 7-11上傳列印串接
    * PDF/CSV匯出
    * 健保網頁查詢
    * Swagger API 支援
    * 支援多藥品公司查詢
    * Freenom
    * NO-IP
    * Cloudflare
